# Catalog Extensions Test Case Workbook

This folder now includes an Excel workbook for catalog extension user-journey test coverage:

- `catalog_extensions_user_journey_test_cases.xlsx`

Workbook structure:

- `Summary`: scope, assumptions, and coverage totals
- `Storefront Journeys`: browse, filter, search, PDP, listing cards, and catalog UX
- `Checkout and Portal`: cart, simple checkout, order placement, success redirect, tracking, cancel, return, refund
- `Admin and Background`: setup, mappings, validation, badge sync, scheduler, and image override behavior

Assumption used:

- "unit test cases" was interpreted as a test-case catalog suitable for QA planning and later automation, organized around real user journeys in `catalog_extensions`
